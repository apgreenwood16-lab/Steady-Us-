Before Session 1

Start without having it all sorted

You do not need perfect spreadsheets, perfect answers, or a fully organised budget before the first call. The goal is simply to arrive ready to talk honestly about what feels unclear, heavy, or repeatedly postponed.

This first step should feel manageable. The package is designed to create clarity from where you are now, not from some ideal starting point.
01

Book your time

Choose a time that works for both of you if possible so the conversation can begin with shared context.

02

Complete the intake form

Share the basics and a few suggested times. The form is there to make the session easier, not to test how organised you are.

03

Keep preparation light

There is no need to build a perfect system in advance. Light reflection is enough.

04

Arrive ready to talk honestly

It helps to know what feels stressful, confusing, or repeatedly delayed, even if the details are still messy.
