# SteadyUs Pre-Session Worksheet

**Version date:** 2026-05-17  
**Audience:** Client delivery source

This worksheet is designed to help you arrive at **Session 1** with a clearer sense of what feels settled, what feels unclear, and what you most want help with. It is not a test, and it does not need to be completed perfectly. A short, honest version is far more useful than an overworked one.

You do not need exact numbers for every answer. If you only have rough impressions or partial information, that is completely fine. The purpose of this worksheet is simply to make the starting picture easier to talk about together.

## How to use this worksheet

Please complete this worksheet before Session 1 if you can. If you are filling it in as a couple, aim for simple and honest answers rather than long explanations. If only one of you completes it first, that is still useful.

| Section | What it helps with |
| --- | --- |
| Your current picture | Brings the overall money situation into view |
| Pressure points | Identifies where money feels heavy, tense, or avoided |
| Shared priorities | Highlights what matters most to improve |
| Session goals | Helps Session 1 focus on the right conversations |

## 2. Your current money picture

Use short phrases if that is easier. The goal is not detail for detail’s sake. The goal is to understand the broad shape of things.

| Prompt | Your notes |
| --- | --- |
| In one sentence, how would you describe your money situation right now? |  |
| What currently feels organised? |  |
| What currently feels disorganised or unclear? |  |
| Are there any regular expenses or commitments that feel heavy right now? |  |
| Do you feel you have a shared picture of your finances, or does one person hold more of it? |  |

## 3. What feels hardest at the moment

This section helps surface the issues that actually create friction in everyday life.

| Prompt | Your notes |
| --- | --- |
| What money topic do you most often postpone? |  |
| What creates the most stress, tension, or uncertainty? |  |
| Is there anything you avoid looking at because it feels messy or emotionally loaded? |  |
| Where do the two of you feel least aligned? |  |
| If money has felt “stuck,” what seems to keep it that way? |  |

## 4. Priorities and hopes

Please focus on what would make the biggest practical difference over the next few months.

| Prompt | Your notes |
| --- | --- |
| What would you most like to feel clearer about after this process? |  |
| What would make money feel calmer at home? |  |
| What decision or area would you most like help making progress on? |  |
| What would a “good outcome” from these sessions look like for you? |  |

## 5. Practical preparation

| Item | Status / notes |

## 6. Anything important we should know before Session 1

Use this space for context that would help the session feel more useful, more sensitive, or more realistic.

| Prompt | Your notes |
| --- | --- |
| Is there any recent change affecting your finances? |  |

## Final reminder

Please do not worry if some sections are incomplete. A partial worksheet is still useful. The aim is simply to make the starting point easier to see, so Session 1 can focus on what matters most.
