# COVER SHEET — SteadyUs File Categories

This folder has been cleaned so the **main level** is easier to use.

The files now fall into three practical categories.

| Category | What it means | Typical files in this category | When to use it |
| --- | --- | --- | --- |
| **Sales** | Files used to explain, pitch, or summarize the offer before delivery starts. | `SALES - SteadyUs Full Offer Deck.pdf`, `SALES - SteadyUs Prospect Deck.pdf`, `SALES - Couples Budget Reset At A Glance.pdf` | Use these when you are introducing the offer, sending overview material, or having a sales conversation. |
| **Client Delivery** | Files that are suitable to share with clients during the delivery journey. | `CLIENT DELIVERY DECK - SteadyUs Couples Budget Reset.pdf`, `CLIENT DELIVERY - SteadyUs Client Session Guide.pdf` | Use these when you want a client to understand what will happen, what to expect, and how the process works. |
| **Internal Only** | Files for your own use or for another coach delivering the offer. These are not client-facing. | `INTERNAL ONLY DECK - SteadyUs Coach Delivery Guide.pdf`, `INTERNAL ONLY - SteadyUs Coach Delivery Playbook.pdf`, `INTERNAL ONLY - SteadyUs Coach Call Notes Form.pdf` | Use these when preparing delivery, running sessions, taking notes, or training another coach. |

There are also two supporting types of file in this folder.

| Supporting type | What it is for |
| --- | --- |
| **Source files** | The labeled `SOURCE` Markdown files are the editable originals behind the polished PDFs. Use them only when you want to change the wording. |
| **Navigation / package files** | `START_HERE_WHICH_FILE_TO_USE.md`, `VISUAL INDEX - SteadyUs File Map.pdf`, `README.md`, and the two ZIP files help you navigate or share the pack more easily. |

The older unlabeled versions have been moved into `ARCHIVE - Older Unlabeled Files` so the main folder is cleaner while still keeping the earlier files available if you ever need them.
