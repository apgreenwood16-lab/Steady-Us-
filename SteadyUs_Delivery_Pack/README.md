# SteadyUs Delivery Pack

This folder contains the consolidated delivery materials for the **Couples Budget Reset**.

| File | Audience | Purpose |
| --- | --- | --- |
| `SESSION_DELIVERY_PLAN.md` | Coach / internal use | Original Markdown playbook for how to run Session 1 and Session 2. |
| `CLIENT_SESSION_CHECKLIST.md` | Client-facing | Original Markdown orientation guide for couples. |
| `COACH_CALL_NOTES_TEMPLATE.md` | Coach / internal use | Original Markdown note-taking template for live calls. |
| `client_session_guide.html` | Client-facing | Polished handout version of the client journey and session expectations. |
| `coach_delivery_playbook.html` | Coach / internal use | Polished playbook version of the delivery framework for consistent facilitation. |
| `coach_call_notes_form.html` | Coach / internal use | Polished working note sheet for session delivery and follow-up capture. |

The Markdown files remain the editable source versions. The HTML files are the more polished delivery-pack versions.
